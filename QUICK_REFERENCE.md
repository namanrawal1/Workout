## **🐛 IF SOMETHING GOES WRONG**

### Build Failed?
```
Check:
✓ package.json in root folder
✓ All file names exact (case-sensitive)
✓ Folder structure correct
✓ No missing files
```

### App Not Loading?
```
Try:
1. Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
2. Clear cache: Settings → Clear browsing data
3. Try incognito/private mode
4. Wait 2 minutes for deployment to finish
```

### Changes Not Showing?
```
Do:
1. Make change on GitHub
2. Wait 30 seconds
3. Hard refresh browser
4. Check Vercel dashboard (is it green?)
```
